# ExamMate AI 🎯

ExamMate AI is a web-based application that helps students generate exam-oriented notes using AI.

## 🚀 Features
- Login UI
- Dashboard
- AI Notes Generator
- Responsive Design

## 🛠 Tech Stack
- React (Vite)
- Tailwind CSS
- Node.js (future)
- AI Integration (planned)
